How to Connect to the Server

To connect to the server you will need 3 things:

  1. An internet connection with port 22 or 3000 open
  2. An SSH client
    • openSSH
    • PuTTY
    • mobaXterm
    • web-client
  3. A computer

Internet Connection

It is crucial that you have an internet connection with ports 22 or 3000 open. For most networks these ports are open by default, but if your system administrator is strict on what content you can access from your internet connection you will not be able to access the server. This means that most school or work networks are out of the picture.

SSH Clients

There are many SSH clients out there for you to use, but it depends on your needs.

openSSH-client

As of a recent Microsoft Windows update the openSSH client is now pre-installed on the windows command line, this is not the only place it is found though, the openSSH client has been pre-installed on macOS since the beginning and is pre-installed on almost all linux distributions.

Pros

  • pre-installed on most operating systems.

Cons

  • not graphical, command line only.

PuTTY

PuTTY is a free and open source, cross platform SSH client that is available on macOS, Linux, and Windows. It has a graphical interface and does support X-forwarding for graphical programs (if you are running an X server)

Pros

  • runs on almost every operating system.

Cons

  • the program must be downloaded to be run.

Download PuTTY

mobaXterm

mobaXterm is a great client that has many... many... features. The best being an X server for windows. This means that you can run any graphical program on windows easily.

Pros

  • runs an X server for graphical programs.

  • it is a graphical program, very easy to use.

Cons

  • must be downloaded to be run.

  • not open-source, and is insecure.

  • payed for monthly, but there is a freeware version.

  • only available on windows.

Download mobaXterm

Web Client

The web client is by far the easiest way to access the server. Anyone with a javascript-enabled web browser can access the server in their own web browser.

Pros

  • runs on all operating systems.

  • doesn't require anything to install or download.

Cons

  • very hard to forward graphical programs.
